# 我的编程之旅（网页版）

一个能在浏览器里点点鼠标、写写代码就能学 Python 的网页应用，从 `print()` 开始，
一路学到能看懂 `ai-games` 文件夹里的4个AI小游戏。

## 怎么打开

双击 `start.command` 这个文件，会自动打开浏览器并启动本地服务。
（第一次打开有安全提示的话，右键点它选"打开"就行）

如果双击不好使，也可以在终端里手动运行：

```
cd ~/Projects/learn-python/code-course
python3 -m http.server 8765
```

然后浏览器打开 http://localhost:8765/

第一次加载会花几秒钟（要下载浏览器里的Python运行环境），之后会快很多。

## 关卡说明

一共12关：从 `print()`、变量、if判断、循环、列表、字典、函数，
到最后自己写出一个"猜数字"小游戏，并对照着去看 `../ai-games/` 里真正的AI小游戏代码。

写好代码点"运行"，右边会显示输出结果和过关提示；卡住了点"💡提示"。
进度会自动保存在浏览器里，关掉重开也不会丢。
