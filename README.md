# 我的编程之旅（网页版）

一个能在浏览器里点点鼠标、写写代码就能学 Python 的网页应用。
打开后先选择自己的水平，再进入对应的路线，一路学到能看懂 `ai-games` 文件夹里的4个AI小游戏。

## 怎么打开

双击 `start.command` 这个文件，会自动打开浏览器并启动本地服务。
（第一次打开有安全提示的话，右键点它选"打开"就行）

如果双击不好使，也可以在终端里手动运行：

```
cd ~/Projects/learn-python/code-course
python3 -m http.server 8765
```

然后浏览器打开 http://localhost:8765/

第一次加载会花几秒钟（要下载浏览器里的Python运行环境），之后会快很多。

## 两条路线

打开后 (`index.html`) 会先问"你已经会一点编程了吗"：

- **编程新手** → `course.html`，12关，从 `print()`、变量、if判断、循环、列表、字典、函数，
  一路学到自己写出一个"猜数字"小游戏。每一关只给代码骨架和注释提示，要自己写出正确代码才能解锁下一关；
  写错了会根据具体的报错类型给出中文解释。
- **已经有基础** → `assessment.html`，跳过讲解，直接给6道综合测试题
  （FizzBuzz、列表统计、字符串处理、质数判断函数、字典词频统计、预测下一步），
  用来快速检验水平，最后一题的思路和 `ai-games/3_rps_ai.py` 完全对应。

两条路线的学习进度分开保存，互不影响。

## 关卡说明

写好代码点"运行"，右边会显示输出结果和过关提示；卡住了点"💡提示"。
进度会自动保存在浏览器里，关掉重开也不会丢。

## 文件说明（给你未来的自己看）

- `index.html` — 入口选择页
- `course.html` — 新手12关课程页面
- `assessment.html` — 有基础的人的测试页面
- `style.css` — 样式
- `levels.js` — 新手课程每一关的讲解、代码骨架、过关条件
- `assessment-levels.js` — 测试题的题目和过关条件
- `app.js` — 让页面动起来的逻辑，负责在浏览器里真的运行Python代码（course/assessment 共用同一份，用 `TRACK` 变量区分存档）
