# 我的编程之旅（网页版）

一个能在浏览器里点点鼠标、写写代码就能学 Python 的网页应用。
打开后先做一个水平测试，再进入正好适合的起点：初级 → 进阶 → 高级，最后去玩 `ai-games.html` 里的4个AI小游戏。

## 怎么打开

双击 `start.command` 这个文件，会自动打开浏览器并启动本地服务。
（第一次打开有安全提示的话，右键点它选"打开"就行）

如果双击不好使，也可以在终端里手动运行：

```
cd ~/Projects/learn-python/code-course
python3 -m http.server 8765
```

然后浏览器打开 http://localhost:8765/

第一次加载会花几秒钟（要下载浏览器里的Python运行环境），之后会快很多；
如果加载失败或者超时，会自动显示错误提示和"重新加载"按钮。

## 入口：水平测试

打开 `index.html` 会先做选择题（题库44道，每次随机抽10道，不用等Python环境加载，秒开），
覆盖 print、变量、运算、字符串、input、if判断、for/while循环、列表、字典、函数这些概念。

- 全部答对 → 推荐直接去 `assessment.html` 做进阶题目
- 从某一题开始答错 → 推荐从 `course.html?start=N` 对应的那一关开始
  （更前面的关卡会在侧栏自动标记为"⏭️ 测试跳过"，不会跟真正写代码通过的"✅"混在一起）

想跳过测试，也可以在测试首页直接点"直接当初级学"/"直接做进阶"/"直接做高级"。
如果浏览器里已经有学习进度，首页会在测试上方显示"继续上次的学习"入口。

## 四个页面

- **`course.html`（🌱 初级）**：12关，从 `print()`、变量、if判断、循环、列表、字典、函数，
  一路学到自己写出一个"猜数字"小游戏。每一关只给代码骨架和注释提示，要自己写出正确代码才能解锁下一关；
  写错了会根据具体的报错类型给出中文解释，写对了会有彩纸庆祝动画，并且可以直接点"下一关"。
- **`assessment.html`（⚡ 进阶）**：跳过讲解，直接给6道综合测试题
  （FizzBuzz、列表统计、字符串处理、质数判断函数、字典词频统计、预测下一步）。
  答错的话会有链接直接跳回初级对应的关卡复习。
- **`advanced.html`（🚀 高级）**：教初级和进阶都没教过的更硬核内容——
  递归、冒泡排序、凯撒密码、类(class)、自己实现二分法猜数字算法、改进版猜拳AI（只看最近3次）。
- **`ai-games.html`（🎮 AI小游戏）**：真·交互式地玩 `ai-games/` 文件夹里那4个完整程序
  （AI猜数字、聊天机器人、会学习的石头剪刀布、迷你神经元）。点"开始游戏"后跟着提示一步步输入回答，
  代码默认收起来，点"查看代码"才展开；玩到一半可以"清空对话"但不打断游戏进度。

四个页面的学习进度分开保存，互不影响。侧栏顶部有进度显示（"已完成 X / N 关"），
最下面有"重置进度"按钮，可以随时清空重新开始。侧栏顶部可以直接切换四个页面或重新测试。

## 手机适配

窄屏幕下侧栏会变成顶部横条，关卡列表变成可以左右滑动的一排，代码框和输出框上下堆叠。

## 文件说明（给你未来的自己看）

- `index.html` + `quiz.js` + `quiz-app.js` — 入口的水平测试
- `course.html` + `levels.js` — 初级12关
- `assessment.html` + `assessment-levels.js` — 进阶6题
- `advanced.html` + `advanced-levels.js` — 高级6关
- `ai-games.html` + `ai-games-levels.js` + `ai-games-app.js` — AI小游戏网页版（独立的交互引擎，不跟其他页面共用 app.js）
- `style.css` — 样式
- `app.js` — 让页面动起来的逻辑，负责在浏览器里真的运行Python代码。
  初级/进阶/高级三个页面共用同一份，用 `TRACK` 变量区分存档；支持 `?start=N`（测试推荐的起点，前面关卡标记为"测试跳过"）
  和 `?resume=N`（继续上次学习，不影响解锁状态）两种跳转方式。
